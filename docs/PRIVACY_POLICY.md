# QuickTranslate Privacy Policy

Last updated: 2026-05-11

## Overview

QuickTranslate is a browser extension that translates webpage text on demand. The extension processes page text only when the user triggers translation or enables automatic translation for supported pages.

## Data We Process

QuickTranslate may process the following data:

- Page text selected for translation.
- Extension settings stored locally in the browser, such as translation provider, target language, display preferences, glossary, custom style, site rules, and whether automatic translation is enabled.
- A user-provided DeepSeek API key, if the user chooses the DeepSeek provider.
- Cached translation results stored locally in the browser to improve performance.

## How Data Is Used

QuickTranslate uses data only to provide translation features:

- Page text is sent to the selected translation provider to generate translated output.
- Settings are stored in `chrome.storage.local` on the user's device.
- Cached translations are stored locally and automatically expired after a limited retention period.

## Third-Party Services

Depending on the user's selected provider, QuickTranslate may send text to:

- Google Translate public endpoint: `https://translate.googleapis.com/`
- DeepSeek API: `https://api.deepseek.com/`

QuickTranslate does not operate these services. Their processing is governed by their own terms and privacy policies.

## Data Storage

QuickTranslate does not run its own backend and does not transmit data to any server controlled by the extension publisher.

Data stored locally in the browser may include:

- User preferences
- DeepSeek API key
- Temporary translation cache

Users can clear cached translations from the extension options page.

## Data Sharing

QuickTranslate does not sell user data and does not share user data with advertisers or data brokers.

Data is shared only with the translation provider selected by the user and only for the purpose of translation.

## User Control

Users can:

- Choose the translation provider
- Disable automatic translation
- Remove or replace their DeepSeek API key
- Clear cached translation data
- Uninstall the extension at any time
